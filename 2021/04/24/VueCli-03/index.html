VueCli入门之三：style

发表于 2021-04-24 更新于 2021-04-25 分类于 VueCli ， 入门

本章对 VueCli 单文件组件中的 style 标签进行简单说明

style 标签

style 标签内包含当前组件内所需的样式

属性

scoped

scoped 属性可以使当前 style 标签内的样式只作用于其所在的组件内
这意味着一旦使用了 scoped 属性，当前 style 标签内的样式将不会影响全局样式，也不能更改其子组件的样式

lang

lang 属性可以指定一种 css预处理语言 ，所谓 css预处理语言 ，是在原 css 基础上进行的一种扩展，旨在优化其语法，提高开发体验。
目前我们最常用的 css预处理语言 是 Sass ，在我们所开发过前端项目中， package.json 文件中都有一项 node-sass， syle 标签默认支持原生 css 语法，如果希望使用 Sass ，则需要将 lang 属性指定为 scss
