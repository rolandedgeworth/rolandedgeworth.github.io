script 的基本结构

基础应用

本章仅说明基本的 data、method 及部分生命周期函数的使用

name

当前组件的名称，仅在某些特殊情况下有作用，如 keep-alive 时作为唯一标识，有兴趣请自行了解

data

data，即当前组件的状态 / 属性 / 变量

在一个组件中，只要声明了 data ，就必须在其中 return 一个对象
一般来说，我们将临时且局部的数据在 data 中进行声明，将其作为一个基本的变量使用

methods

一个组件中所定义的方法，全部存放于 methods 中

使用 this.函数名() 进行调用

生命周期钩子

对于生命周期，可以将其理解为一个 Vue 实例 从创建到销毁的整个过程，而生命周期钩子，则是在各个具有代表性的时间节点上所执行的方法（函数）
