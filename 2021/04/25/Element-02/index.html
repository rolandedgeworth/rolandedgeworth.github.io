使用 Element 渲染一个表格，首先需要指定表格整体所对应的数据，目前只接收 Array 类型，其它格式不可。在上文的示例中，有 :data="tableData"，这一命令将变量 tableData 绑定到了一个 el-table 组件上，简而言之，这将使这一 el-table 显示变量 tableData 中的数据

绑定好数据后，需要指定表格的每一列

el-table-column ，由语义可知，这一组件为 Element 表格的一个列，其基本参数有 prop 与 label ，分别对应这一列绑定的字段名（tableData 中遍历每一个对象 key 值）与这一列所显示的表头文本

根据当前主流的业务需求，一个合格的表格，至少要有：

  1. 对其中各个列的宽度自适应，且可以手动调整
  2. 每一列的不同显示方式，从普通文本到点击弹窗，抑或动态编辑
  3. 对数据过长，或是列数过多，导致一屏无法完全展示的应对方案
  4. 单行或多行的选择，及其对应的钩子

我们将根据这四项需求进行讲解

需求类型 1 ：width

固定宽度 width，单位 px，为当前列指定一个固定不变的宽度

最小宽度 min-width，单位同样为 px，为当前列指定最小宽度，在整体宽度足够时会自动加宽

需求类型 2 ：slot

在你明白何为作用域插槽前，记住这五行代码，将其理解为一段确定的，不可更改的语法即可

在一个列标签中，插入了一个 template 标签，其属性 slot-scope 的取值为 "scope"

在这一 template 标签内部，你可以对这一列做任何期望的处理，在这里可以使用 v-if 选择性地渲染不同的节点（如不同颜色的小 tag），也可使用过滤器，或者向某一方法传递参数……总之，在这一 template 标签内，可以通过 scope.row 取到遍历渲染列表时的每一列数据
