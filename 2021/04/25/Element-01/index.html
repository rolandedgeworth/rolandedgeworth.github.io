发表于 2021-04-25 分类于 VueCli ， Element UI

本章对 Element UI 进行简单介绍

UI 框架

框架（framework），即某种应用的半成品，提供常用且基本性的功能

一个成熟的 UI 框架，应至少包括表格（Table）、表单（Form）、弹窗（Popup）及其相关功能，它应该具有对主流浏览器的兼容能力，但跨平台性并不在我们的考虑范围内

在日常前端开发工作中，UI 框架于前端开发人员而言，最大的意义是简化了数据与 DOM 的交互操作，且保证其健壮性，进而大幅度提高前端开发效率

目前我们最常用的 UI 框架是 Element UI，本章选择几项最常用功能进行简介

Element UI

前文提到，一个成熟的 UI 框架，应至少包括表格（Table）、表单（Form）、弹窗（Popup）及其相关功能，而这正是我们最常用，且最为耗神的功能，在这里只对这三种组件进行说明
